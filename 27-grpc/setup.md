# How to setup a workplace

1. install protoc from https://github.com/protocolbuffers/protobuf/releases
2. go install google.golang.org/protobuf/cmd/protoc-gen-go@latest
3. go install google.golang.org/grpc/cmd/protoc-gen-go-grpc@latest
4. make sure go\bin in the PATH
5. install make if not found. (for win, its simpler to run commands manually then istall make)
